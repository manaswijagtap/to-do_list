# To-Do List Application (JavaFX)

## Overview
A desktop-based To-Do List application built using JavaFX, Maven, and SQLite.
It allows users to create, update, delete, and persist daily tasks efficiently.

## Features
- Add, update, and delete tasks
- Local data persistence using SQLite
- Clean JavaFX UI
- MVC architecture

## Tech Stack
- Java
- JavaFX
- Maven
- SQLite

## How to Run
1. Clone the repository
2. Open in IntelliJ / Eclipse
3. Build using Maven
4. Run Main.java
